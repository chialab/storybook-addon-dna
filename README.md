# Storybook Addon DNA

A storybook addon for better [DNA](https://www.chialab.it/p/dna) components integration.

---

## Install

```sh
$ npm i @chialab/storybook-addon-dna -D
$ yarn add @chialab/storybook-addon-dna -D
```
